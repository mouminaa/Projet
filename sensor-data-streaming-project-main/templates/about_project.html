Работа с датчиками IoT Первая часть состоит из платформы, которая эмулирует датчики произвольной природы. Может настраиваться частота обновления, тип, вероятность "всплеска" обновлений. Project Task (English Translation)

Working with IoT sensors The first part consists of a platform that emulates sensors of arbitrary nature. The update frequency, type, and probability of "burst" updates can be configured. Readings are recorded in MQTT, Kafka, etc. The second part of the system processes the readings, finds anomalies over specified time intervals, and stores history. Graphs are displayed in the UI.

About the Project

Created by Takudzwa Mupanesure, this project presents a cutting-edge solution for monitoring and visualizing IoT sensor data. At its core, the project consists of several key components:

  • Sensor Data Emulation: Simulates various types of IoT sensors, allowing for flexible configuration of update frequencies and types.
  • Real-Time Data Processing: Manages the continuous stream of data from the emulated sensors, ensuring timely processing and analysis.
  • Anomaly Detection: Utilizes the Isolation Forest machine learning algorithm, a part of the scikit-learn library, to effectively identify anomalies in sensor data. This model excels in detecting outliers, making it ideal for real-time monitoring.
  • Data Visualization: Interactive charts and graphs in the user interface provide a dynamic and intuitive representation of both real-time and historical sensor data.
  • Apache Kafka: Implemented using Docker, Apache Kafka serves as a robust backbone for handling large-scale data streams. Kafka's distributed architecture, combined with Docker's containerization, offers a scalable and reliable platform for data ingestion and processing.
  • Dockerization: The use of Docker containers for deploying Apache Kafka and other components of the system ensures a seamless and consistent environment. This approach simplifies deployment, enhances portability, and aids in maintaining consistent performance across different environments.

This project represents a comprehensive approach to IoT data handling, from collection and processing to visualization, all backed by the efficiency and reliability of modern technologies like Kafka and Docker.
